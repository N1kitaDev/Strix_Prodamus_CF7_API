# CF7 Prodamus XL Integration

Плагин для интеграции Contact Form 7 с API Prodamus XL. Автоматически создает лиды и сделки в Prodamus XL при отправке форм Contact Form 7.

## Возможности

- Автоматическая интеграция с Contact Form 7
- Создание лидов и сделок в Prodamus XL
- Настраиваемые параметры API
- Поддержка нескольких форм
- Логирование всех операций

## Установка

1. Скачайте и активируйте плагин
2. Убедитесь, что Contact Form 7 установлен и активирован
3. Перейдите в админку: **CF7 Prodamus XL**
4. Введите ваш API токен из Prodamus XL
5. Настройте остальные параметры (при необходимости)
6. Укажите ID форм для интеграции (или оставьте пустым для всех форм)

## Настройка полей формы

Плагин автоматически ищет следующие поля в формах Contact Form 7:

- `your-name` - Имя контакта (обязательно)
- `your-email` - Email контакта (обязательно)
- `your-phone` - Телефон контакта
- `your-contactway` - Предпочитаемый способ связи

## Настройки

### API Token
Ваш токен авторизации из Prodamus XL. Можно найти в настройках аккаунта.

### Stage ID
ID этапа воронки продаж, где будут создаваться сделки. По умолчанию: `1gKg6-JsY0aftO_lcO6vwQ` ("Новая регистрация")

### Responsible User ID
ID пользователя, ответственного за сделки. По умолчанию: `GEqhVZNU5kG2dgZf48RkEQ`

### Form IDs
Список ID форм Contact Form 7 через запятую. Если оставить пустым, интеграция будет работать для всех форм.

## Логирование

Все операции логируются в файл отладки WordPress. Проверьте логи в случае проблем:
- `wp-content/debug.log`

## Тестирование

1. Заполните форму Contact Form 7 с необходимыми полями
2. Отправьте форму
3. Проверьте логи на наличие ошибок
4. Проверьте создание лида и сделки в Prodamus XL

## Поддержка

При возникновении проблем:
1. Проверьте логи WordPress
2. Убедитесь, что API токен корректный
3. Проверьте настройки формы Contact Form 7
4. Убедитесь, что все необходимые поля присутствуют в форме

## Совместимость

- WordPress 5.0+
- Contact Form 7 5.0+
- PHP 7.0+
